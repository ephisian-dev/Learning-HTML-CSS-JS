## Active Recall

**Q1: What is the purpose of `<!DOCTYPE html>` and how does it influence how a browser displays a webpage?**

`<!DOCTYPE html>` defines the HTML version to the browser so it knows
how to render the page correctly.

---

**Q2: Which types of elements belong in `<head>` vs `<body>`, and why is `<title>` placed in `<head>`?**

The `<head>` stores metadata the user doesn't see directly — charset,
language, title. The `<body>` contains everything the user actually
sees on the page.

---

**Q3: What is the difference between `<ul>` and `<ol>`?**

`<ul>` is an unordered list — bullet points, no required order.
`<ol>` is an ordered list — numbered from 1, used for steps or instructions
where sequence matters.

---

**Q4: What do `<em>` and `<strong>` represent, and what happens when you nest them inside another element?**

`<em>` makes text italic and signals emphasis. `<strong>` makes text bold
and signals importance. When nested inside another element like `<li>`,
only the wrapped text is affected — the rest of the element is unchanged.
