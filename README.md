# Learning HTML, CSS & JS

Self-teaching web development from scratch with zero prior knowledge.
Started April 2026, Year 9.

---

## Goal

Build a fully self-hosted portfolio website from raw HTML, CSS and JS —
no templates, no frameworks. Hosted on Oracle Cloud via Nginx.
This repo documents every step of getting there.

---

## Roadmap

| Phase               | Focus                                 |
|---------------------|---------------------------------------|
| HTML foundations    | Structure, semantics, forms, media    |
| CSS                 | Styling, Flexbox, Grid, animations    |
| JavaScript          | DOM, events, logic, APIs              |
| Portfolio build     | Combining everything into a real site |
| Oracle Cloud deploy | Linux server, Nginx, HTTPS, domain    |

---

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Oracle Cloud (Ubuntu + Nginx) — for portfolio deployment
